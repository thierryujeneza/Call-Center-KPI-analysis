Featured Project

Call Center KPI Analysis

Key finding: Extended calls consistently produced lower CSAT scores — with an overall FCR rate of 74.49%, average CSAT of 3.90, and average handle time of 7.5 minutes across 294 cases. Henry Bizimana led all agents at 85.71% FCR. Validated independently across SQL, Python, and Power BI.

Tools used

Microsoft Excel SQLite DB Browser Python 3 pandas seaborn matplotlib openpyxl Power BI Desktop

What was done

01 Excel — PROPER() name standardisation, IFERROR mean imputation, COUNTIF duplicate detection
02 SQL — 52 queries using RANK(), HAVING, CTEs, subqueries, CREATE VIEW for reusable reporting
03 Python — pandas groupby aggregations, seaborn visualisations, correlation matrix confirming findings
04 Power BI — 15+ DAX measures, slicers, dual-axis trend chart for executive dashboard

HR Attrition Analysis

Key finding: R&D experienced the highest attrition volume at 282 employees, while the overall attrition rate stood at 17% across 2,925 total employees — with male attrition (63%) significantly outpacing female (37%).
